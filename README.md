# 🌤️ JS Weather App

A clean, responsive weather app built with vanilla HTML, CSS, and JavaScript. No frameworks, no API keys — just open it and go.


## ✨ Features

- **Live weather data** — current conditions, 24-hour hourly forecast, and 7-day daily forecast
- **Auto location** — detects your location via GPS with one click
- **City search** — search any city in the world
- **°C / °F toggle** — switch temperature units instantly
- **Dynamic sky themes** — background changes based on conditions (sunny, cloudy, rainy, snowy, night)
- **Animated effects** — drifting blobs, twinkling stars at night, rain and snow particles
- **Hourly scroll buttons** — navigate the 24-hour forecast with prev/next arrows
- **Wind compass** — visual compass showing real wind direction
- **UV index bar** — color-coded UV level indicator
- **Sunrise & sunset times** — with day length calculation
- **Fully responsive** — works on mobile and desktop

## 🌐 APIs Used

| Service | Purpose | Cost |

| [Open-Meteo](https://open-meteo.com) | Weather forecast data | Free, no key |
| [Open-Meteo Geocoding](https://open-meteo.com/en/docs/geocoding-api) | City name → coordinates | Free, no key |
| [Nominatim (OpenStreetMap)](https://nominatim.org) | GPS coordinates → city name | Free, no key |

No API keys, no accounts, no sign-ups required.
